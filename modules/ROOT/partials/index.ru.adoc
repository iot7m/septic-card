*Septic Card* — это кастомная карточка для *Home Assistant*, предназначенная для визуализации и мониторинга уровня заполнения септика.

Карточка предназначена для наглядного и компактного отображения данных датчиков, используемых в системе мониторинга септика.

Карточка работает с агрегированной sensor-entity и отображает:

- текущий уровень жидкости
- настроенные пороговые и критические значения
- дополнительные параметры, такие как температура и давление (при наличии)
- аварийные состояния и ошибки

В данной документации описаны установка карточки, настройка сущностей и рекомендации по её использованию в интерфейсе Home Assistant.
