= История изменений
:description: История изменений GSeptik Card — новые возможности, улучшения и исправления для Home Assistant.

:navtitle: Версия v0.0.2

Версия **v0.0.2** является первой рабочей версией проекта и отражает этап активного прототипирования. В рамках данного релиза была сформирована базовая архитектура карточек, реализована поддержка визуального редактора Home Assistant, а также заложена основа для дальнейшего развития функциональности и расширения тестового покрытия.

Changelog:

- реализована карточки Cistern
- обновлён контракт карточки и схема параметров
- выполнен рефакторинг дизайна и стилей карточек
- переработан механизм вычисления критического уровня
- добавлена поддержка визуального редактора Home Assistant (Card Editor)
- удалены жёстко заданные (hardcoded) сенсоры
- добавлены source maps для dev сборки
- добавлены тесты для карточек, редактора и диалогов
- исправлены ошибки форматирования и линтинга
- подготовлена базовая инфраструктура автоматической сборки и CI
- добавлена начальная документация и usage-раздел в README
- добавлена генерация документации Typedoc
