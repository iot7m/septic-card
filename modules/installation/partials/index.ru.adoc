Раздел описывает способы установки *Septic Card* в системе *Home Assistant*.

Карточка распространяется как кастомный фронтенд-компонент и подключается к интерфейсу Home Assistant через механизм ресурсов Lovelace.

== Установка через HACS

Поддержка установки через *Home Assistant Community Store (HACS)* находится в разработке и на данный момент недоступна.

Рекомендуется использовать ручной способ установки.

== Ручная установка

Для ручной установки необходимо скачать файл карточки и добавить его в список ресурсов Home Assistant.

Сначала загрузите последнюю версию файла `septic-card.js` со страницы релизов проекта и скопируйте его в каталог Home Assistant:

```
/config/www/septic/septic-card.js
```

После этого добавьте карточку как ресурс одним из способов, описанных ниже.

=== Добавление ресурса через интерфейс

1. Перейдите в *Настройки* → *Панели* → *Ресурсы*.
2. Нажмите *Добавить ресурс*.
3. Укажите путь к файлу:
`/local/septic/septic-card.js`
4. Выберите тип ресурса *JavaScript Module*.

=== Добавление ресурса через YAML

Добавьте ресурс в конфигурацию Lovelace:

```yaml
resources:
  - url: /local/septic/septic-card.js
    type: module
```

После добавления ресурса обновите страницу браузера или очистите кэш, если карточка не появилась сразу.

После установки карточка становится доступной для добавления в дашборды Home Assistant.
